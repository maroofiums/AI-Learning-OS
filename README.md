# AI Learning OS

A long-term personal platform unifying learning, research, note-taking, AI assistance,
project management, and knowledge management into one system.

This is not a portfolio project or a clone of an existing tool - see
`docs/` for the full philosophy, architecture, and roadmap.

## Documentation

- `docs/ARCHITECTURE.md` - system architecture and component overview
- `docs/PROJECT_ROADMAP.md` - phase-by-phase build plan (8-12 months)
- `docs/ARCHITECTURE_DECISIONS.md` - finalized technical decisions (ADRs)
- `docs/ER_DIAGRAM.md` - database schema and entity relationships
- `docs/CODING_STANDARDS.md` - enforceable coding rules and definition of done
- `docs/PHASE0_SUMMARY.md` - Phase 0 completion checklist

## Status

**Phase 0 - Planning: complete.**
Next: Phase 1 - Foundation (FastAPI structure, auth, PostgreSQL, Alembic, Docker).

FastAPI - PostgreSQL - SQLAlchemy 2.0 (async) - Alembic - Redis - Celery - FAISS to Qdrant - Next.js - React - Tailwind CSS
